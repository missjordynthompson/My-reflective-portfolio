Cloud Computing


About Cloud Computing

Cloud Computing is a type of computing that's used
to store, manage, and process data, that delivers
convenient, and on-demand access over the internet.
There are different types of clouds:

  • Public cloud - these are shared, easily accessible
    clouds that provide applications, storage or other
    services over the internet, for example
    google, amazon etc.
  • Private cloud - these are clouds that are only
    accessible by one person or an organisation, for
    example datacom, spark etc.
  • Hybrid clouds - these are a mix of public
    and private clouds
  • Vertical cloud - these allow customers to build
    different applications with special specifications
    and needs for different sectors of their business.

    (Rainer, Gray, & Pye, 2014). There are lots of benefits
    to cloud computing to both individuals and groups
    including flexibility, disaster recovery, easy access,
    work from anywhere, and environmentally friendly etc.
    (Salesforce UK, 2015). The cloud has increasingly
    becoming so popular that 50% of businesses are
    expected to completely move to the public cloud in the
    next couple of years (Bommadevara, Kaplan, &
    Starikova, 2016). However there are disadvantages of
    the cloud such as internet access is required, access
    can be impaired by something being damaged or by
    man-made/natural disasters, privacy and security
    issues as the internet and computers are never and
    could never be private (Wink, 2012).





    • My Personal Reflection

    Cloud computing has changed the way most students
    my age study. I use cloud computing all the time and I
    upload all my lecture notes, code, images, and other
    personal things to the public cloud (google). This is
    because its easy to access from anywhere I go, I can
    control who accesses certain files, I can work on
    group projects no matter where everyone is, and it is
    just incredibly easy when you have to constantly move
    from class to class. The only downside I have had so
    far would be that if the internet fails (for example at
    university), I was unable to access any files what so
    ever and was stuck for hours until the internet was
    fixed. Although this is the only bad personal
    experience I have had so far, the positives outweigh
    the bad and I still continue to the public cloud for
    everything I do.
